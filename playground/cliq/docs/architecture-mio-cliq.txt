  Identity rules (enforced in REST client):
    ─ Channel post  ?bot_unique_name=tobytime  → speaks AS bot
    ─ Reactions / DMs via /buddies              → speaks AS OAuth user (hard wall)
    ─ DM-as-bot only via Bot Message Handler    return Map (sync reply)

  Auth header (NOT Bearer):    Authorization: Zoho-oauthtoken <access_token>
  Webhook signature header:    X-Webhook-Signature: sha256=<base64|hex>
